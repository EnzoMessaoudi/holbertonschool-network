## What is localhost/127.0.0.1  
It is the standard adress for IPV4 loopback trafic  

## What is 0.0.0.0  
It is the source address used during the initialization process  

## What is /etc/hosts  
It is the hosts file that navigator use to translate hostnames into IP address. When we research a page in the navigator, this one will look into the hosts file, get the ip address and connect to the right server.
